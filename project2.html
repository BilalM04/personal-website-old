Summary

Design Scenario:

In Project Two, our team was tasked with creating a system for securely transferring surgical instruments to an autoclave for sterilization. There were two sub-teams for this project: the modelling sub-team and the computing sub-team. The modelling team was responsible for designing a sterilization container to hold the surgical instruments. The computing sub-team, which I was a part of, was responsible for creating a Python program to control the robotic arm.

Project Tasks:

  • Determine the objectives, constraints, and functions of a sterilization container and perform a morphological analysis.
  • Create a flowchart for the various functions required for the program.
  • Convert the flowcharts to pseudocode.
  • Code the functions in the Quanser virtual environment.
  • Conduct a materials research summary to determine a suitable material for the sterilization container.
  • Present the program to an instructional assistant intern during a project interview.
  • Summarize the project in a final design report.
